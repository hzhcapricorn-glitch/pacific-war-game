# 卡牌名称换行功能

## 功能说明

为了让长卡牌名称在UI中更美观，实现了自动换行功能。

## 实现方式

### 1. CSS支持

在 `cards.css` 中添加 `white-space: pre-line`：

```css
.card-name {
  white-space: pre-line; /* 支持 \n 换行 */
}

.detail-name {
  white-space: pre-line; /* 支持 \n 换行 */
}
```

**white-space: pre-line 作用：**
- 保留 `\n` 换行符
- 自动折叠连续空格
- 允许文本自动换行

### 2. JSON数据格式

在卡牌名称中使用 `\n` 换行符：

```json
{
  "name": "法拉格特级\n驱逐舰"
}
```

## 技术细节

### CSS属性对比

| 属性 | 效果 | 是否使用 |
|------|------|---------|
| `white-space: normal` | 忽略 `\n`，自动换行 | ❌ |
| `white-space: pre` | 保留 `\n`，不自动换行 | ❌ |
| `white-space: pre-line` | 保留 `\n`，自动换行 | ✅ |
| `white-space: pre-wrap` | 保留所有空格和 `\n` | ❌ |

**选择 pre-line 的原因：**
- ✅ 保留手动 `\n` 换行
- ✅ 折叠多余空格
- ✅ 自动换行长文本
- ✅ 不影响现有布局

### JSON转义规则

在JSON中：
- `\n` → 换行符
- `\\n` → 字面字符串 `\n`（错误）

**正确写法：**
```json
"name": "法拉格特级\n驱逐舰"
```

**错误写法：**
```json
"name": "法拉格特级\\n驱逐舰"  // 显示为 "法拉格特级\n驱逐舰"
```

### 战斗报告
战斗报告中的卡牌名称会自动使用换行后的名称：

```
战损情况:
  - 法拉格特级

```

如果不希望在报告中换行，可以在生成报告时替换：
```javascript
const reportName = card.name.replace(/\n/g, '');
```

## 添加新卡牌

添加需要换行的新卡牌时：

1. **直接在JSON中使用 \n：**
   ```json
   {
     "name": "新型级\n战列舰"
   }
   ```

2. **或使用脚本更新：**
   - 编辑 `update_card_names.py`
   - 添加到 `name_replacements` 字典
   - 运行脚本

## 注意事项

1. **不要过度换行**：每张卡牌最多2行
2. **保持一致性**：同类型卡牌使用相同换行规则
3. **测试显示**：确保不同视图下都显示正常
4. **战斗报告**：确认多行名称在报告中可读
