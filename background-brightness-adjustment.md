# 背景亮度调整说明

## 问题诊断

原始配置导致背景灰蒙蒙的原因：
1. **商店区域**原始背景色是黑色半透明：`rgba(0, 0, 0, 0.15)`
2. **白色覆盖层**太弱：`rgba(255, 255, 255, 0.05)` 只有 5% 透明度
3. 缺少基础的浅色背景衬托

## 调整方案

### 1. 更换原始背景色（从暗到亮）

| 区域 | 原始背景色 | 新背景色 | 说明 |
|------|-----------|---------|------|
| 商店区域 | `rgba(0, 0, 0, 0.15)` 黑色 | `rgba(255, 255, 255, 0.08)` 白色 | 从暗色改为浅色 |
| 部署区 | `rgba(59, 130, 246, 0.1)` 深蓝 | `rgba(147, 197, 253, 0.15)` 浅蓝 | 更亮的蓝色 |
| 手牌区 | 无 | `rgba(255, 255, 255, 0.12)` 白色 | 新增浅色背景 |

### 2. 增强白色覆盖层（提升亮度）

从 `0.05` → `0.12` (提升140%)

**效果对比：**
```css
/* 调整前 - 灰蒙蒙 */
background-color: rgba(255, 255, 255, 0.05);  /* 太弱 */

/* 调整后 - 更明亮 */
background-color: rgba(255, 255, 255, 0.12);  /* 增强亮度 */
```

### 3. 背景图透明度保持不变

保持 `opacity: 0.15` - 让背景图适度显示，不抢眼

## 最终配置总览

### 手牌区 (`.hand-zone`)
```css
background-color: rgba(255, 255, 255, 0.12);  /* 主背景 */

::before {
  background-image: url('Background_hand.png');
  opacity: 0.15;  /* 背景图 */
}

::after {
  background-color: rgba(255, 255, 255, 0.12);  /* 白色覆盖层 */
  backdrop-filter: blur(2px);
}
```

### 部署区 (`.deployed-zone`)
```css
background-color: rgba(147, 197, 253, 0.15);  /* 浅蓝色主背景 */

::before {
  background-image: url('Background_deploy.png');
  opacity: 0.15;
}

::after {
  background-color: rgba(255, 255, 255, 0.12);
  backdrop-filter: blur(2px);
}
```

### 商店区域 (`.shop-section-left/right`)
```css
background-color: rgba(255, 255, 255, 0.08);  /* 浅色主背景 */

::before {
  background-image: url('Background_shop1/2.png');
  opacity: 0.15;
}

::after {
  background-color: rgba(255, 255, 255, 0.12);
  backdrop-filter: blur(2px);
}
```

## 调整效果

**调整前：**
- ❌ 灰蒙蒙，缺乏明亮感
- ❌ 黑色底色吸收光线
- ❌ 白色覆盖层太弱

**调整后：**
- ✅ 明亮清爽
- ✅ 浅色底色反射光线
- ✅ 白色覆盖层增强，提升整体亮度
- ✅ 背景图依然柔和不突出

## 进一步调整建议

如果还想更亮，可以：

1. **提高主背景亮度**
   ```css
   background-color: rgba(255, 255, 255, 0.15);  /* 从 0.08 → 0.15 */
   ```

2. **增强白色覆盖层**
   ```css
   background-color: rgba(255, 255, 255, 0.18);  /* 从 0.12 → 0.18 */
   ```

3. **降低背景图透明度**（让它更显眼）
   ```css
   opacity: 0.2;  /* 从 0.15 → 0.2 */
   ```

⚠️ **注意：** 过亮可能导致卡牌文字对比度下降，需要平衡视觉效果和可读性。

---

**推荐配置（当前）：**
- 主背景：白色 0.08-0.12
- 覆盖层：白色 0.12
- 背景图：15% 透明度
- 模糊：2px

这个配置在亮度和柔和度之间取得了良好平衡。
