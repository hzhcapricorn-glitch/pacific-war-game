# 太平洋反击战 - 游戏规则

## 游戏背景

第二次世界大战，太平洋战场。1941年12月7日，珍珠港遭受日本偷袭，美国太平洋舰队损失惨重。作为美军指挥官，你需要重建海军力量，通过一系列海战、空战和登陆作战，逐步反击日本，最终取得太平洋战争的胜利。

## 游戏目标

完成所有任务卡牌，最终赢得冲绳战役，取得游戏胜利。

## 游戏流程

每个回合由以下6个阶段组成：

### 1. 准备阶段（Prepare）
- **自动执行**：所有部署区的横置卡牌自动恢复为直立状态
- **商店刷新**：随机商店自动刷新，未购买的随机卡牌返回牌堆，重新抽取6张展示
- 玩家无需操作，系统自动进入下一阶段

### 2. 抽卡阶段（Draw）
- **自动执行**：从抽牌堆抽取5张卡牌到手牌
- 如果抽牌堆卡牌不足，自动将弃牌堆洗牌后作为新的抽牌堆
- 玩家无需操作，系统自动进入下一阶段

### 3. 行动阶段（Action）
- **玩家操作**：从手牌中打出卡牌
- 点击手牌中的卡牌即可使用
- **战术卡**使用后直接进入弃牌堆，不会进入部署区
- 部署区中横置的**单位/后勤卡**可以通过支付"重置费用"来重新激活
- **界面提示**：本阶段商店区域会灰显（但仍可悬停查看卡牌详情）
- 完成所有操作后，点击"下一阶段"按钮

### 4. 战斗阶段（Combat）
- **玩家操作**：选择部署区的卡牌参与战斗
- 点击部署区的卡牌来选择/取消选择（被选中的卡牌会高亮显示）
- 只有**单位卡**可以参加战斗，**后勤卡**不能参战
- 空军单位需要航空槽位支持（由航空母舰等单位提供）
- 选择完成后，点击"开始战斗"进行战斗结算
- 也可以选择"跳过战斗"直接进入下一阶段

### 5. 购买阶段（Shop）
- **玩家操作**：使用补给点数在商店购买卡牌
- **商店布局**：
  - **左侧 - 必要卡牌**：固定展示的核心卡牌（如补给舰、航空母舰），可重复购买，库存堆叠显示
  - **右侧 - 随机卡牌**：每回合刷新的6张随机卡牌，不堆叠，每张只能购买一次
- 点击卡牌进行购买，购买的卡牌会进入弃牌堆
- **补给显示**：商店右上角显示当前补给和最大保留（格式：`X / Y`）
- **界面提示**：本阶段部署区会灰显（但仍可悬停查看卡牌详情）
- 如果有剩余补给且商店中还有可购买的卡牌，切换阶段时会弹出确认提示
- 完成购买后，点击"下一阶段"按钮

### 6. 弃牌阶段（Discard）
- **自动执行**：所有手牌自动弃掉，进入弃牌堆
- 超过最大保留上限的补给会被清除
- 玩家无需操作，系统自动回到准备阶段，开始新的回合

## 基础规则

### 补给系统
- **补给**：游戏中的货币，用于购买卡牌和重置横置的单位
- **获取方式**：通过打出提供补给的卡牌（如基础补给、补给舰等）
- **显示位置**：商店右上角显示"补给: X / Y"（当前补给 / 最大保留）
- **最大补给保留**：每回合结束时，只能保留一定数量的补给，超过部分会被清除
- **提升上限**：通过某些卡牌的能力可以提升最大补给保留（如补给舰）

### 卡牌类别
- **战术卡**：立即生效后直接进入弃牌堆，不会进入部署区，没有重置费用
  - 例如：基础补给（提供1点补给）
  - 例如：退役（从弃牌堆移除一张卡牌）
- **单位卡**：可以参加战斗的单位，提供火力支援和特殊能力
  - 例如：战斗机、轰炸机、驱逐舰、巡洋舰、战列舰、航空母舰、海军陆战队
  - 航空母舰可以提供航空槽位，让空军单位参战
- **后勤卡**：提供资源和能力，但不能参加战斗
  - 例如：补给舰（提供补给和增加补给上限）

### 卡牌稀有度系统

游戏中的卡牌根据强度和数量分为4个稀有度等级：

- **N（普通）**：基础卡牌，商店中数量充足（≥8张）
  - 边框颜色：灰色
  
- **R（稀有）**：优质卡牌，商店中数量中等（5-7张）
  - 边框颜色：蓝色
  
- **SR（超稀有）**：强力卡牌，商店中数量稀少（2-4张）
  - 边框颜色：紫色
  
- **UR（终极稀有）**：最强卡牌，商店中仅有1张
  - 边框颜色：金黄色

### 单位类型
- **空军**：战斗机、轰炸机等，需要航空槽位才能参战
- **海军**：驱逐舰、巡洋舰、战列舰、航空母舰等
- **陆军**：海军陆战队等登陆作战单位

### 商店系统

商店分为两个独立的区域，提供不同的购买体验：

#### 必要卡牌区（左侧）
- **特点**：核心卡牌，始终可购买
- **展示方式**：堆叠显示，右上角显示剩余数量
- **购买机制**：可重复购买同一类型卡牌，库存由卡牌稀有度决定

#### 随机卡牌区（右侧）
- **特点**：每回合变化，增加游戏策略性
- **展示方式**：不堆叠，平铺显示6张卡牌
- **刷新机制**：
  - 每回合准备阶段自动刷新
  - 未购买的卡牌返回随机牌堆
  - 从随机牌堆中随机抽取6张新卡牌展示
- **购买机制**：每张卡牌只能购买一次，购买后从商店移除
- **随机卡牌**：包括所有战斗单位，不同稀有度的卡牌在牌堆中的数量不同
  - N卡（8张）：F4F野猫、弗莱彻级驱逐舰等
  - R卡（5-7张）：SBD无畏、巴尔的摩级巡洋舰、海军陆战队等
  - SR卡（3-4张）：B-25米切尔轰炸机等
  - UR卡（1张）：衣阿华级战列舰

#### 商店刷新
- **自动刷新**：每回合准备阶段自动执行
- **手动刷新**：可使用Debug面板的"刷新商店"按钮立即刷新随机商店

### 火力系统
游戏中有三种火力类型：

- **对地火力**：用于登陆作战和岛屿攻击
- **对海火力**：用于海战，攻击敌方舰船
- **对空火力**：用于空战和防空

不同的单位具有不同的火力组合。例如：
- 战斗机：高对空火力，低对海火力
- 轰炸机：高对地/对海火力，低对空火力
- 战列舰：高对海火力，中等对地/对空火力

### 航空槽位系统
- **航空槽位**：由航空母舰等单位提供
- **空军限制**：参战的空军单位数量不能超过可用的航空槽位总数
- **计算方式**：选中的所有单位提供的航空槽位之和
- 如果航空槽位不足，系统会阻止战斗并提示玩家调整

### 战斗系统

#### 战斗准备
1. 在战斗阶段，选择要参战的卡牌
2. 只能选择直立状态（未横置）的卡牌
3. 后勤卡不能参战
4. 确保空军单位数量不超过航空槽位

#### 战斗结算
1. 计算己方的三种火力总和
2. 与当前任务的火力需求对比：
   - **对地火力** ≥ 任务需求对地火力
   - **对海火力** ≥ 任务需求对海火力
   - 同时满足以上两个条件即为**胜利**

3. **对空火力的特殊规则**：
   - 对空火力**不影响胜负**
   - 但如果对空火力**不足**，战斗损失会**加倍**
   - 例如：任务要求损失1张卡牌，对空不足时会损失2张

#### 战斗奖励与损失
- **胜利**：获得任务卡上的奖励（补给、最大补给保留等）
- **失败或胜利**：根据任务卡的损失条件，随机损失一定数量的参战卡牌
- **对空不足**：损失数量加倍
- **损失卡牌的去向**：
  - **必要卡牌**（如补给舰、航母）：返回必要商店，可重新购买
  - **随机卡牌**（如战斗机、驱逐舰）：返回随机商店牌堆，可能在后续回合刷新时出现
  - **战术卡**：不会损失（战术卡使用后直接进弃牌堆，不参战）

#### 战斗后
- 所有参战的卡牌会横置
- 横置的卡牌在下一回合的准备阶段自动恢复
- 或在行动阶段支付重置费用来重新激活

## 名词解释

### 卡牌区域
- **抽牌堆**：待抽取的卡牌，显示剩余数量
- **手牌**：当前可以使用的卡牌
- **部署区**：已使用的单位卡牌所在区域
- **弃牌堆**：已使用完的卡牌，点击可以查看所有弃牌
- **商店**：分为两个区域
  - **必要商店**（左侧）：固定展示的核心卡牌，堆叠显示库存
  - **随机商店**（右侧）：每回合刷新的随机卡牌
- **随机商店牌堆**：不可见的随机卡牌池，准备阶段从中抽取卡牌到随机商店

### 卡牌状态
- **直立**（Ready）：可以使用/参战的状态
- **横置**（Tapped）：已经使用过，暂时无法参战的状态

### 卡牌属性
- **稀有度**：N/R/SR/UR，决定商店库存数量和卡牌边框颜色
- **类别**：战术/单位/后勤，决定卡牌的使用方式
- **单位类型**：空军/海军/陆军，决定单位的作战特点
- **价格**：在商店购买此卡牌需要的补给
- **重置费用**：在行动阶段重新激活横置卡牌需要的补给（战术卡无此属性）
- **对地/对海/对空火力**：该单位在战斗中提供的火力值
- **航空槽位**：该单位可以搭载的空军数量（仅航空母舰等单位拥有）
- **商店类型**：
  - **必要卡牌**：始终在商店中可购买
  - **随机卡牌**：每回合随机出现在商店中

### 任务属性
- **需要对地火力**：完成任务需要的最低对地火力
- **需要对海火力**：完成任务需要的最低对海火力
- **需要对空火力**：达不到时损失加倍，但不影响胜负
- **奖励**：战斗胜利后获得的奖励
- **损失**：战斗后损失的卡牌数量

## 游戏提示

### 策略建议
1. **平衡发展**：注意对地、对海、对空火力的均衡发展
2. **航母优先**：从必要商店尽早购买航空母舰，解锁空军单位（SR稀有度，仅4张库存）
3. **补给管理**：注意最大补给保留上限，及时购买补给舰提升上限（R稀有度，7张库存）
4. **后勤保障**：后勤卡虽然不能参战，但提供重要的资源支持
5. **损失控制**：注意对空火力要求，避免损失加倍
6. **商店策略**：
   - 必要卡牌库存根据稀有度设定，关键卡牌及时购买
   - 随机商店每回合刷新，遇到强力卡牌及时抓住机会
   - 每回合查看随机商店，规划购买优先级
   - **稀有度关注**：
     - UR卡（金色边框）：整个牌堆只有1张，极其珍贵，遇到立即考虑购买
     - SR卡（紫色边框）：2-4张，相对稀少，优先级较高
     - R卡（蓝色边框）：5-7张，中等稀有度，根据需求购买
     - N卡（灰色边框）：8张以上，数量充足，随时可以购买
7. **战术卡运用**：合理使用退役卡，移除弱卡优化牌组
8. **稀有卡牌收藏**：高稀有度卡牌通常具有更强的属性或独特能力，值得优先收集

### 常见问题
- **Q: 为什么我的空军不能参战？**
  - A: 需要先部署航空母舰等提供航空槽位的单位

- **Q: 为什么补给舰不能选中参战？**
  - A: 补给舰属于后勤卡，不能参加战斗，但可以在部署区提供持续效果

- **Q: 战斗失败了，卡牌损失去哪了？**
  - A: 必要卡牌返回必要商店，随机卡牌返回随机牌堆（后续可能刷新出来）

- **Q: 对空火力不够会怎样？**
  - A: 对空火力不足不影响胜负，但会导致战斗损失加倍

- **Q: 随机商店什么时候刷新？**
  - A: 每回合准备阶段自动刷新，也可以使用Debug面板的"刷新商店"按钮手动刷新

- **Q: 为什么有些卡牌打出后没有出现在部署区？**
  - A: 战术卡（如基础补给、退役）使用后直接进入弃牌堆，不会进入部署区

- **Q: 非购买阶段商店和部署区是灰色的，怎么办？**
  - A: 这是正常的阶段限制，防止误操作。但你仍然可以悬停查看卡牌详细信息

- **Q: 补给显示在哪里？**
  - A: 商店右上角显示"补给: X / Y"（当前补给 / 最大保留）

- **Q: 退役卡怎么使用？**
  - A: 在行动阶段从手牌打出退役卡，会弹出窗口让你选择要从弃牌堆移除的卡牌，选择后该卡牌永久移除游戏

- **Q: 为什么随机商店的卡牌不堆叠？**
  - A: 随机卡牌每张都是独立的实例，购买后即移除，不像必要卡牌有库存概念

- **Q: 卡牌边框颜色代表什么？**
  - A: 边框颜色代表稀有度：灰色=N（普通），蓝色=R（稀有），紫色=SR（超稀有），金黄色=UR（终极稀有）

- **Q: 稀有度影响卡牌强度吗？**
  - A: 稀有度主要影响商店中的库存数量。通常高稀有度卡牌有更强属性或独特能力，但N/R卡牌也有其战术价值

- **Q: 如何找到UR卡？**
  - A: UR卡在整个随机商店牌堆中只有1张，需要在每回合的商店刷新中碰运气。看到金色边框的卡牌就是UR卡

## 界面说明

### 信息栏（顶部）
- **回合数**：当前回合
- **阶段**：当前游戏阶段
- **Debug按钮**：
  - **补给+10**：增加10点补给（测试用）
  - **刷新商店**：立即刷新随机商店
  - **重新开始**：重启游戏

### 商店区域（上方）
- **补给显示**：右上角显示"补给: X / Y"
- **必要卡牌**：左侧堆叠显示，右上角显示库存数量，边框颜色表示稀有度
- **随机卡牌**：右侧平铺显示，每回合刷新，边框颜色表示稀有度
- **稀有度识别**：通过卡牌边框颜色快速识别稀有度（灰/蓝/紫/金）
- **灰显状态**：非购买阶段会灰显，但仍可悬停查看详情

### 部署区（中间）
- **直立卡牌**：可参战或使用能力
- **横置卡牌**：已使用，需要重置
- **选中状态**：战斗阶段选中的卡牌会高亮显示
- **灰显状态**：购买阶段会灰显，但仍可悬停查看详情

### 手牌区（下方）
- **点击使用**：在行动阶段点击打出
- **战术卡**：打出后直接进弃牌堆

### 弃牌堆
- **点击查看**：点击可以查看所有弃牌
- **显示数量**：只显示卡牌数量，不展示具体卡牌

### 详细信息面板（右侧）
- **悬停显示**：鼠标悬停在任何卡牌上可查看详情
- **稀有度显示**：R/SR/UR卡牌会在详细信息中显示稀有度标签
- **固定高度**：面板固定高度，防止战场简讯位置跳动

### 战场简讯（右下）
- **实时日志**：显示游戏事件和行动记录
- **最多20条**：自动保留最近20条记录
