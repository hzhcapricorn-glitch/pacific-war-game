# 太平洋反击战 - 游戏规则

## 游戏背景

第二次世界大战，太平洋战场。1941年12月7日，珍珠港遭受日本偷袭，美国太平洋舰队损失惨重。作为美军指挥官，你需要重建海军力量，通过一系列海战、空战和登陆作战，逐步反击日本，最终取得太平洋战争的胜利。

## 游戏目标

完成所有任务卡牌，最终赢得冲绳战役，取得游戏胜利。

## 游戏流程

每个回合由以下6个阶段组成：

### 1. 准备阶段（Prepare）
- **自动执行**：所有部署区的横置卡牌自动恢复为直立状态
- 玩家无需操作，系统自动进入下一阶段

### 2. 抽卡阶段（Draw）
- **自动执行**：从抽牌堆抽取5张卡牌到手牌
- 如果抽牌堆卡牌不足，自动将弃牌堆洗牌后作为新的抽牌堆
- 玩家无需操作，系统自动进入下一阶段

### 3. 行动阶段（Action）
- **玩家操作**：从手牌中打出卡牌
- 点击手牌中的卡牌即可使用
- 部署区中横置的卡牌可以通过支付"重置费用"来重新激活
- 完成所有操作后，点击"下一阶段"按钮

### 4. 战斗阶段（Combat）
- **玩家操作**：选择部署区的卡牌参与战斗
- 点击部署区的卡牌来选择/取消选择（被选中的卡牌会高亮显示）
- 只有**战斗卡**和**支援卡**可以参加战斗，**后勤卡**不能参战
- 空军单位需要航空槽位支持（由航空母舰等单位提供）
- 选择完成后，点击"开始战斗"进行战斗结算
- 也可以选择"跳过战斗"直接进入下一阶段

### 5. 购买阶段（Shop）
- **玩家操作**：使用补给点数在商店购买卡牌
- 点击商店中的卡牌堆进行购买
- 购买的卡牌会进入弃牌堆
- 如果有剩余补给且商店中还有可购买的卡牌，切换阶段时会弹出确认提示
- 完成购买后，点击"下一阶段"按钮

### 6. 弃牌阶段（Discard）
- **自动执行**：所有手牌自动弃掉，进入弃牌堆
- 超过最大保留上限的补给会被清除
- 玩家无需操作，系统自动回到准备阶段，开始新的回合

## 基础规则

### 补给系统
- **补给**：游戏中的货币，用于购买卡牌和重置横置的单位
- **获取方式**：通过打出提供补给的卡牌
- **最大补给保留**：每回合结束时，只能保留一定数量的补给，超过部分会被清除
- **提升上限**：通过某些卡牌的能力可以提升最大补给保留

### 卡牌类别
- **补给卡**：使用后直接进入弃牌堆，不会进入部署区（如：基础补给）
- **战斗卡**：可以参加战斗的单位，提供火力支援
- **支援卡**：可以参加战斗，同时提供特殊能力（如：航空母舰）
- **后勤卡**：提供资源和能力，但不能参加战斗（如：补给舰）

### 单位类型
- **空军**：战斗机、轰炸机等，需要航空槽位才能参战
- **海军**：驱逐舰、巡洋舰、战列舰、航空母舰等
- **陆军**：海军陆战队等登陆作战单位

### 火力系统
游戏中有三种火力类型：

- **对地火力**：用于登陆作战和岛屿攻击
- **对海火力**：用于海战，攻击敌方舰船
- **对空火力**：用于空战和防空

不同的单位具有不同的火力组合。例如：
- 战斗机：高对空火力，低对海火力
- 轰炸机：高对地/对海火力，低对空火力
- 战列舰：高对海火力，中等对地/对空火力

### 航空槽位系统
- **航空槽位**：由航空母舰等单位提供
- **空军限制**：参战的空军单位数量不能超过可用的航空槽位总数
- **计算方式**：选中的所有单位提供的航空槽位之和
- 如果航空槽位不足，系统会阻止战斗并提示玩家调整

### 战斗系统

#### 战斗准备
1. 在战斗阶段，选择要参战的卡牌
2. 只能选择直立状态（未横置）的卡牌
3. 后勤卡不能参战
4. 确保空军单位数量不超过航空槽位

#### 战斗结算
1. 计算己方的三种火力总和
2. 与当前任务的火力需求对比：
   - **对地火力** ≥ 任务需求对地火力
   - **对海火力** ≥ 任务需求对海火力
   - 同时满足以上两个条件即为**胜利**

3. **对空火力的特殊规则**：
   - 对空火力**不影响胜负**
   - 但如果对空火力**不足**，战斗损失会**加倍**
   - 例如：任务要求损失1张卡牌，对空不足时会损失2张

#### 战斗奖励与损失
- **胜利**：获得任务卡上的奖励（补给、最大补给保留等）
- **失败或胜利**：根据任务卡的损失条件，随机损失一定数量的参战卡牌
- **对空不足**：损失数量加倍
- 损失的卡牌会返回商店，可以重新购买

#### 战斗后
- 所有参战的卡牌会横置
- 横置的卡牌在下一回合的准备阶段自动恢复
- 或在行动阶段支付重置费用来重新激活

## 名词解释

### 卡牌区域
- **抽牌堆**：待抽取的卡牌
- **手牌**：当前可以使用的卡牌
- **部署区**：已使用的单位卡牌所在区域
- **弃牌堆**：已使用完的卡牌，可以点击查看
- **商店**：可以购买的卡牌池

### 卡牌状态
- **直立**（Ready）：可以使用/参战的状态
- **横置**（Tapped）：已经使用过，暂时无法参战的状态

### 卡牌属性
- **价格**：在商店购买此卡牌需要的补给
- **重置费用**：在行动阶段重新激活横置卡牌需要的补给
- **对地/对海/对空火力**：该单位在战斗中提供的火力值
- **航空槽位**：该单位可以搭载的空军数量

### 任务属性
- **需要对地火力**：完成任务需要的最低对地火力
- **需要对海火力**：完成任务需要的最低对海火力
- **需要对空火力**：达不到时损失加倍，但不影响胜负
- **奖励**：战斗胜利后获得的奖励
- **损失**：战斗后损失的卡牌数量

## 游戏提示

### 策略建议
1. **平衡发展**：注意对地、对海、对空火力的均衡发展
2. **航母优先**：尽早购买航空母舰，解锁空军单位
3. **补给管理**：注意最大补给保留上限，及时提升上限
4. **后勤保障**：后勤卡虽然不能参战，但提供重要的资源支持
5. **损失控制**：注意对空火力要求，避免损失加倍

### 常见问题
- **Q: 为什么我的空军不能参战？**
  - A: 需要先部署航空母舰等提供航空槽位的单位

- **Q: 为什么补给舰不能选中参战？**
  - A: 补给舰属于后勤卡，不能参加战斗，但可以在部署区提供持续效果

- **Q: 战斗失败了，卡牌损失去哪了？**
  - A: 损失的卡牌返回商店，可以重新购买

- **Q: 对空火力不够会怎样？**
  - A: 对空火力不足不影响胜负，但会导致战斗损失加倍
