# 有效成本公式优化分析

## 公式变更

### 旧公式（已废弃）
```
e_cost = (cost + rd_cost × 2)²
```

**问题：**
- 过度惩罚整备成本（2倍系数 + 平方）
- 未能反映购买难度的真实增长曲线
- 对高价卡的惩罚过重

### 新公式（当前使用）
```
e_cost = cost^1.4 + rd_cost × cost^0.6 × 1.5
```

**优势：**
- 捕捉购买难度的**超线性增长**（指数1.4）
- 整备成本影响随卡牌价格适度增长（cost^0.6）
- 更符合游戏实际体验

## 理论基础

### 1. 购买难度的非线性增长

**卡组配置：** 5张+1，2张+2，1张+3补给，10张其他卡，每轮抽5张

**每轮期望补给：**
```
E = 5 × [(5/18 × 1) + (2/18 × 2) + (1/18 × 3)]
  = 5 × 12/18 ≈ 3.33点/轮
```

**实际购买所需回合数：**

| 卡牌价格 | 期望回合数 | 累积难度 |
|---------|-----------|---------|
| 1-2     | 1轮       | 基准    |
| 3-4     | 1-2轮     | 中等    |
| 5-6     | 2轮       | 较高    |
| 7-8     | 2-3轮     | 很高    |

**结论：** 购买难度不是线性的，但也不是平方关系。指数1.4是一个合理的平衡点。

### 2. 整备成本的复合影响

整备成本 `rd_cost` 的影响：

1. **直接影响：** 占用 rd_cost 轮，卡牌无法使用
2. **机会成本：** 整备期间无法产生价值
3. **预算压力：** 整备期间可能错过购买机会

**为什么使用 `cost^0.6` ？**
- 高价卡的整备成本影响更严重（失去的机会价值更大）
- 但不是线性关系（0.6 < 1.0）
- 避免过度惩罚

## 参数调优建议

### 如果补给更充足（如改为10张补给卡）

购买难度降低，可以降低指数：
```python
e_cost = cost^1.3 + rd_cost × cost^0.6 × 1.5
```

### 如果游戏节奏更快（整备影响更大）

提高整备惩罚系数：
```python
e_cost = cost^1.4 + rd_cost × cost^0.6 × 2.0
```

### 如果想更精确

使用分段函数：
```python
if cost <= 3:
    base = cost × 1.2
elif cost <= 5:
    base = 3.6 + (cost - 3) × 1.5
else:
    base = 6.6 + (cost - 5) × 2.0

e_cost = base + rd_cost × cost × 0.6
```

## 公式数学推导

### 为什么是 1.4？

假设每轮期望补给为 E，购买价格为 C 的卡牌需要 n 轮：

```
n ≈ C / E
```

但实际上由于方差和累积机制，真实回合数为：

```
n_real ≈ (C / E)^α
```

其中 α 取决于补给分布。对于你的卡组配置，通过模拟可得 α ≈ 1.1-1.2。

考虑到还有其他隐性成本（如购买时机、手牌管理等），最终选择 **1.4** 作为指数。

### 为什么是 cost^0.6？

整备成本的影响与卡牌价值相关。高价卡在整备期间失去的机会价值更大。

但这个影响不是线性的，因为：
1. 低价卡更容易替代
2. 高价卡通常效果更强，整备后回报更高

经验值 **0.6** 在 [0.5, 0.8] 区间内表现最佳。
