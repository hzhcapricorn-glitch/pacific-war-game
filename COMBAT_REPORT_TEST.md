# 战斗报告测试文档

## 新功能概述

战斗报告现在会显示详细的损伤情况：

1. **重甲单位**：显示被重创次数及是否击沉
2. **空军单位**：显示是否成功返航或坠毁
3. **普通单位**：正常显示损失

## 战斗报告格式

### 示例1：重甲单位未被击沉

```
🎉 战斗胜利！

己方火力:
  对地: 10 / 8
  对海: 10 / 8
  防空: 7 / 5 ✓
  制空: 4 / 3 ✓ 空军可返航

获得奖励:
  - 补给 +3

战损情况:
  - 北卡罗来纳级战列舰 被重创1次但未被击沉 🛡️
  - 法拉格特级驱逐舰 损失
```

**说明**：战列舰（重甲1）被选中1次，不足2次，未被击沉，返回弃牌堆。

### 示例2：重甲单位被击沉

```
🎉 战斗胜利！

己方火力:
  对地: 16 / 10
  对海: 16 / 10
  防空: 8 / 8 ✓
  制空: 2 / 5

获得奖励:
  - 补给 +5

战损情况:
  - 衣阿华级战列舰 被重创2次并击沉 🛡️💥
  - 巴尔的摩级巡洋舰 损失
  - 海军陆战队 损失
```

**说明**：衣阿华级（重甲1）被选中2次，达到击沉条件，返回商店。

### 示例3：空军返航成功

```
🎉 战斗胜利！

己方火力:
  对地: 8 / 5
  对海: 12 / 8
  防空: 9 / 6 ✓
  制空: 6 / 5 ✓ 空军可返航

获得奖励:
  - 补给 +4

战损情况:
  - F6F地狱猫战斗机 被击中但成功返航 ✈️
  - SBD无畏式俯冲轰炸机 被击中但成功返航 ✈️
```

**说明**：制空火力充足（6 ≥ 5），空军单位触发返航能力，进入弃牌堆而非商店。

### 示例4：空军坠毁

```
❌ 战斗失败...

己方火力:
  对地: 5 / 8
  对海: 6 / 10
  防空: 8 / 6 ✓
  制空: 2 / 5

战损情况:
  - F4F野猫战斗机 被击中并坠毁 ✈️💥
  - TBD鱼雷机"破坏者" 被击中并坠毁 ✈️💥
```

**说明**：制空火力不足（2 < 5），空军单位无法返航，返回商店需重新购买。

### 示例5：防空不足，损失加倍

```
🎉 战斗胜利！

己方火力:
  对地: 10 / 8
  对海: 12 / 10
  防空: 3 / 8 ⚠️ 不足！损失加倍
  制空: 3 / 6

获得奖励:
  - 补给 +5

战损情况:
  - 北卡罗来纳级战列舰 被重创2次并击沉 🛡️💥
  - 法拉格特级驱逐舰 损失
  - 弗莱彻级驱逐舰 损失
  - 海军陆战队 损失
```

**说明**：
- 原本损失2张，防空不足加倍为4张
- 战列舰在4次随机选择中被选中2次，达到击沉条件
- 其他3张卡牌被选中各1次，全部损失

### 示例6：复杂场景（重甲+空军+普通）

```
🎉 战斗胜利！

己方火力:
  对地: 15 / 12
  对海: 15 / 12
  防空: 10 / 8 ✓
  制空: 7 / 6 ✓ 空军可返航

获得奖励:
  - 补给 +6

战损情况:
  - 衣阿华级战列舰 被重创1次但未被击沉 🛡️
  - F6F地狱猫战斗机 被击中但成功返航 ✈️
  - 巴尔的摩级巡洋舰 损失
```

**说明**：
- 战列舰：被选中1次 < 2次，未击沉，返回弃牌堆
- 空军：制空充足，成功返航，进入弃牌堆
- 巡洋舰：普通单位，正常损失，返回商店

## 测试步骤

### 测试1：基础重甲测试
1. 部署：1个北卡罗来纳级战列舰 + 2个驱逐舰
2. 执行任务：损失=2，防空要求低
3. 预期报告：
   - 战列舰可能显示"被重创1次但未被击沉"
   - 或者"被重创2次并击沉"
   - 驱逐舰正常损失

### 测试2：空军返航成功
1. 部署：2个战斗机 + 1个航母 + 1个驱逐舰
2. 执行任务：损失=2，制空要求≤己方制空
3. 预期报告：
   - 空军显示"被击中但成功返航"
   - 驱逐舰可能损失

### 测试3：空军坠毁
1. 部署：2个战斗机 + 1个航母（但制空不足）
2. 执行任务：损失=2，制空要求>己方制空
3. 预期报告：
   - 空军显示"被击中并坠毁"

### 测试4：防空不足+重甲
1. 部署：1个战列舰 + 2个陆战队
2. 执行任务：损失=2，防空要求高（触发加倍→4）
3. 预期报告：
   - 损失加倍提示
   - 战列舰可能被重创2次击沉（概率较高）

### 测试5：企业号（幸运）+战列舰（重甲）
1. 部署：1个企业号 + 1个战列舰 + 2个驱逐舰
2. 执行任务：损失=3
3. 预期报告：
   - 优先损失驱逐舰
   - 战列舰可能被重创但未击沉
   - 企业号应该最后才被选中

## 图标说明

- 🛡️ - 重甲防护
- 💥 - 击沉/坠毁
- ✈️ - 空军单位
- ✓ - 满足条件
- ⚠️ - 警告（防空不足）

## 注意事项

1. **随机性**：每次战斗的损伤分布不同，需要多次测试观察
2. **概率计算**：
   - 重甲1的单位被击沉概率 = (被选中2次的概率)
   - 参战单位越多，重甲单位被击沉概率越低
3. **显示逻辑**：
   - 只显示受到损伤的单位（hits > 0）
   - 未受损的单位不出现在战损情况中
