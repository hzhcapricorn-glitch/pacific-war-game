# DBG 卡牌构筑游戏

一个基于 Web 的卡牌构筑游戏（Deck-Building Game），灵感来自"巴巴罗萨"。通过 GitHub Pages 托管，可以直接在浏览器中游玩。

## 游戏特性

- 📦 **卡牌构筑机制** - 通过购买和打出卡牌来构建自己的牌组
- ⚔️ **战斗系统** - 选择卡牌参战，完成任务目标
- 💰 **资源管理** - 管理补给资源，平衡购买与打牌
- 🎯 **任务系统** - 完成5个渐进式任务赢得游戏
- 🎨 **现代UI** - 响应式设计，流畅的动画效果
- ⚡ **自动流程** - 准备、抽卡、弃牌阶段自动执行
- 🔍 **弃牌堆查看** - 点击弃牌堆查看所有已弃卡牌

## 游戏玩法

### 基本概念

- **补给**：游戏中的资源，用于购买卡牌和重新部署横置的卡牌
- **横置/重置**：卡牌在使用或战斗后会横置，需要支付费用来重置
- **战斗力**：卡牌的攻击力，用于完成任务

### 游戏流程

每回合分为以下阶段：

1. **准备阶段** ⚡ - 自动重置所有横置的卡牌
2. **抽卡阶段** ⚡ - 自动抽取5张卡牌到手中
3. **行动阶段** 🎮 - 打出手牌，获得补给或其他效果（需要操作）
4. **战斗阶段** 🎮 - 选择卡牌参战，挑战当前任务（需要操作）
5. **购买阶段** 🎮 - 使用补给从商店购买新卡牌（需要操作）
6. **弃牌阶段** ⚡ - 自动弃掉所有手牌，清除超限补给

⚡ = 自动执行阶段 | 🎮 = 需要玩家操作

### 胜利条件

完成所有5个任务卡即可获胜。每个任务需要一定的战斗力才能完成，战斗后会有卡牌损失。

### 最新功能

#### 🆕 智能卡牌系统
- **补给卡优化**：补给卡（如"初级补给"）直接使用后进入弃牌堆，不占用部署区空间
- **自动流程**：准备、抽卡、弃牌阶段自动执行，无需手动点击
- **只在需要时暂停**：行动、战斗、购买阶段等待玩家操作

#### 🆕 牌堆管理
- **弃牌堆查看**：只显示最后一张卡牌，点击可查看所有弃牌
- **抽牌堆隐藏**：只显示数量，保持抽牌的神秘感
- **悬停详情**：所有可见卡牌都可悬停查看完整信息

## 游戏设计

### 卡牌类型

- **战斗卡牌** - 提供战斗力和各种能力
- **任务卡牌** - 游戏的目标，完成后获得奖励

### 卡牌能力

- `supply` - 提供补给
- `draw` - 抽卡
- `max_supply` - 增加最大补给保留
- `combat_boost` - 战斗力加成

### 初始卡牌

游戏开始时，玩家拥有5张"初级补给"卡，每张提供1点补给。

### 可购买卡牌

商店中有多种卡牌可供购买：
- 步兵班、炮兵连、装甲师 等战斗单位
- 补给站、后勤军团 等补给卡
- 侦察单位、特种部队 等功能卡
