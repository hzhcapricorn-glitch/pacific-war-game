## 性价比计算公式

### 有效成本 (e_cost)
```
e_cost = (cost + rd_cost × 2)²
```
考虑整备成本的有效成本。整备成本乘以2是因为卡牌需要回到手牌才能再次使用。

### 1. 纯火力效率 (power_efficiency)
```
格式: "火力/cost | 火力/e_cost"
- 火力/cost = TP / cost
- 火力/e_cost = TP / e_cost
```

### 2. 综合价值效率 (value_efficiency)
```
格式: "价值/cost | 价值/e_cost"
- 价值/cost = TV / cost
- 价值/e_cost = TV / e_cost
```

## 能力价值计算公式

本分析使用以下公式计算能力价值:

- 补给: 值 × 2
- 抽卡: 值 × 3
- 最大补给(储备): 值 × 3
- 退役: 固定值 4
- 幸运(保护): 固定值 5
- 返航: 固定值 2
- 侦查: 值 × 2.5
- 商店扩容: 值 × 4
- 快速整备: 值 × 5
- 航空槽位: 每槽 4
